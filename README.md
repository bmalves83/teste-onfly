# Onfly Teste Bruno Alves (onfly)

Um teste para analisar o nível técnico

## Instalação das dependências

```bash
yarn
# or
npm install
```

### Iniciar o app em ambiente de desenvolvimento

## Iniciar o json-server

```bash
npm run mock
```

## Iniciar o ambiente em desenvolvimento

```bash
npm run dev
```

## Teste

```bash
npm run test
# ou visual
npm run test:ui
```

### Build para produção

```bash
npm run build
```
